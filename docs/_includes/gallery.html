{%- comment -%} Galleria fotografica auto-popolata da una cartella di static_files. Parametri: path (obbligatorio) percorso della cartella, es. /assets/img/eventi/foo/ credits (opzionale) hash filename → photographer_id (vedi _data/photographers.yml) {%- endcomment -%} {%- assign imgs = site.static_files | where_exp: "f", "f.path contains include.path" -%}
{%- for img in imgs -%} {%- assign ext = img.extname | downcase -%} {%- if ext == ".jpg" or ext == ".jpeg" or ext == ".webp" or ext == ".png" -%} {%- assign fname = img.name -%} {%- assign credit_id = include.credits[fname] -%} {%- assign author = site.data.photographers[credit_id] -%} {%- endif -%} {%- endfor -%}
